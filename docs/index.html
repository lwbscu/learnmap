它解决的是“我看过了，但没学会”

Skill 的价值不只是生成内容，而是强制走完从地图、互动、检测到复教的闭环。

HTML lessons

从第 1 课开始就是网页课件

每课都是自包含 HTML，带目录、tabs、折叠深读、即时测验、自检清单和底部导航。

Mastery checks

不会让模糊理解直接进入下一章

测验结果会被诊断为正确、部分正确、概念误区、边界不清或应用脱节。

Learning record

网页行为可以交回给 AI

复制详细 Markdown 报告，或下载 JSON，让下一轮教学知道你错在哪、勾了什么。

Bilingual artifacts

中文模式下连文件名也中文

例如 `学习/强化学习/第01课-全局地图/课件.html`，但 MDP、PPO 等术语保留英文。

Mistake-driven reteach

错题不是结束，而是下一轮教学入口

AI 会读取学习进度、错题记录和导出的学习记录，针对薄弱点补讲和复测。

No build step

安装的是一个 Skill，不是复杂平台

仓库只有 Markdown 指令、引用模板和元数据；课件生成在你的本地学习目录里。

为什么它值得 Star

很多 AI 学习提示词只负责“讲”，这个 Skill 负责把学习过程变成可追踪、可复盘、可继续的系统。
